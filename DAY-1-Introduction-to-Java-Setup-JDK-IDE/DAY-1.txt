<--------------------------------------------INTRODUCTION TO JAVA---------------------------------------------------------------->
1.Java is an object-oriented programming language that is used for developing large-scale applications.
2.Java is a platform-independent language, meaning that it can run on any device that has a Java
3.Virtual Machine (JVM) installed.
4.java is robust , secure, and scalable.
5.java is a high-level language, meaning that it abstracts away many low-level details, making
it easier to write and maintain code.
6.java is a statically-typed language, meaning that the data type of a variable is known at
compile time.
7.java is a multi-paradigm language, meaning that it supports multiple programming paradigms,
such as object-oriented, imperative, and functional programming.
8.java is a widely-used language, with a large community of developers and a wide range of
applications.
9.java is a language that is used for developing Android apps, web applications, desktop
applications, and enterprise software.
10.java is a language that is used for developing games, simulations, and other types of
applications
